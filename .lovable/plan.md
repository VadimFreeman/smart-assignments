

## План: Платформа для учебных заданий

### 1. Главная страница
- Приветственный экран с кнопкой "Войти по коду" для учеников
- Отдельная кнопка "Вход для учителей" (менее заметная)
- Минималистичный, чистый дизайн

### 2. Вход для учеников
- Поле для ввода кода задания
- Поле для ввода имени ученика
- После ввода — переход к заданию

### 3. Страница выполнения задания
- Отображение названия и описания задания
- Для тестов: варианты ответов с возможностью выбора
- Для открытых вопросов: текстовое поле для ответа
- Кнопка "Отправить ответ"
- После отправки: показ результата (для тестов — сразу балл)

### 4. Вход для учителей
- Страница авторизации (email/пароль)
- Тестовый аккаунт: логин `admin@admin.com`, пароль `admin`

### 5. Панель учителя
- Список созданных заданий
- Кнопка "Создать задание"
- Просмотр статистики по каждому заданию

### 6. Создание задания
- Ввод названия и описания
- Выбор типа: тест или открытый вопрос
- Для тестов: добавление вопросов с вариантами ответов и отметка правильного
- Генерация уникального кода для доступа

### 7. Просмотр ответов учеников
- Список учеников, выполнивших задание
- Для тестов: автоматический балл
- Для открытых вопросов: текст ответа
- Возможность поставить оценку и написать комментарий

### 8. База данных (Lovable Cloud)
- Таблица учителей с ролями
- Таблица заданий (название, тип, код доступа, вопросы)
- Таблица ответов учеников (имя, ответы, оценка, комментарий)
